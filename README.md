# MarketCheckoutOOPW3

This project refactors a procedural supermarket checkout program into C++ classes.

## Classes

- Product: stores product id, name, and price
- Catalog: manages product list and product lookup
- Cart: stores items and quantities
- Pricing: calculates subtotal, discount, tax, and total
- ReceiptPrinter: prints the receipt

## Output

The program prints a simple market receipt with subtotal, discount, tax, and total.
